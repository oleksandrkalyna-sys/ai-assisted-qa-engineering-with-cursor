# Add section slide

Insert a branded section header into the appropriate `pages/*.md` file.

## Template

```markdown
---
layout: section
---

# Section title
```

Light variant (optional):

```markdown
---
layout: section
variant: light
---

# Section title
```

## Rules

- Use only `layout: section` (binds `public/brand/bg-section-*.jpg`).
- Place logo/motif via the layout — do not paste raw `<img>` for brand chrome.
- English title only; no timing or facilitator cues.
